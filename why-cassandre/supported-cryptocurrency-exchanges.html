# Supported cryptocurrency exchanges

# Supported by XChange

Cassandre uses XChangeopen in new window, a Java library providing a streamlined API for interacting with 60+ Bitcoin and Altcoin exchanges providing a consistent interface for trading and accessing market data.

You can find hereopen in new window a table showing a list of exchanges and how XChange supports them.

# Validated by Cassandre

Cassandre can theoretically support the 60+ cryptocurrency exchanges the way XChange does but we can't test them all.

For the exchanges providing a sandbox, we were able to create integration tests for :

  • Coinbaseopen in new window: Working except cancel order.
  • Geminiopen in new window: Working but market orders are not supported.
  • Kucoinopen in new window: Working.

We also run real bots with real assets on Kucoinopen in new window, Coinbaseopen in new window and Binanceopen in new window to make sure you can use it safely.
