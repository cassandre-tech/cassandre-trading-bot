# Créez votre project Cassandre

Si vous n'avez pas de projet spring boot existant, vous pouvez utiliser notre archetype mavenopen in new window pour en générer un :

mvn archetype:generate -B \
-DarchetypeGroupId=tech.cassandre.trading.bot \
-DarchetypeArtifactId=cassandre-trading-bot-spring-boot-starter-basic-archetype \
-DarchetypeVersion=6.0.0 \
-DgroupId=com.mycompany.bot \
-DartifactId=my-trading-bot \
-Dversion=1.0-SNAPSHOT \
-Dpackage=com.mycompany.bot

Voici la structure du projet qui va être créé pour vous :

my-trading-bot/
├── pom.xml
└── src
    ├── main
    │   ├── java
    │   │   └── com
    │   │       └── mycompany
    │   │           └── bot
    │   │               ├── Application.java
    │   │               ├── package-info.java
    │   │               └── SimpleStrategy.java
    │   └── resources
    │       └── application.properties
    └── test
        ├── java
        │   └── com
        │       └── mycompany
        │           └── bot
        │               └── SimpleStrategyTest.java
        └── resources
            ├── application.properties
            ├── candles-for-backtesting-BTC-USDT.csv
            ├── user-main.tsv
            ├── user-savings.csv
            └── user-trade.csv
